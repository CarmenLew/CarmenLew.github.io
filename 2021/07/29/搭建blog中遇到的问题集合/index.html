搭建blog的步骤与问题集合

  • 教学视频：https://www.bilibili.com/video/BV1Yb411a7ty?from=search&seid=16485403914348981730

    视频中用mac操作系统教学，流程差不多，cmd输入命令行稍微有所不同，通过弹幕大神可得知win的操作方法
  • 本机之前安装过git。有弹幕反馈：没安装过git的可能会在第5步失败

1.安装node.js，装完用 node -v 和 npm -v 查看版本验证是否成功

2.安装淘宝镜像cnmp

npm install -g cnpm --registry=https://registry.npm.taobao.org

装完查看版本来验证

3.通过cnmp来安装hexo

cnpm install -g hexo-cli

查看版本验证安装是否成功

4.chidir显示当前目录

d:进入D盘

新建目录

由dos时代继承到win console的md/mkdir没有区别，
都是建立新目录make directory的意思
用法 md [盘符:][路径]新目录名
比如md c:\test

进入目录

cd D:\blog

5.hexo init

第一次尝试失败

在弹幕中得知要以管理员身份打开cmd，重来，成功！

用dir 查看该目录下的文件夹,对于mac的ls -l

6. hexo s本地启动hexo

s是server的简写

7.hexo new “我的第一篇博客”

创建了一个md文件

8.博客存放的位置D:\blog\source\_posts,文件格式为md
